Liver Cirrhosis Prediction App

Description: A machine learning-powered web application that predicts liver cirrhosis risk based on patient health metrics. Built using Flask, scikit-learn, and HTML forms, this app enables quick and accessible health risk assessment.

Key Features

  • Flask web server with clean routes
  • Pretrained Random Forest model with normalization pipeline
  • User-friendly form for inputting medical data
  • Prediction result rendering with interpretation

Technologies Used

  • Python, Flask
  • scikit-learn for model training and preprocessing
  • Pickle for model serialization
  • HTML, CSS for UI

Outcome

This project demonstrates how machine learning can be integrated into healthcare web applications for practical use. It showcases Flask routing, model handling, and form processing skills.
